\doxysubsection{Detailed Description}
Application principale pour ESP32-\/\+S3 avec écran tactile C3248\+W535C. 

Cette application implémente un système de navigation multi-\/pages pour l\textquotesingle{}écran tactile QSPI 480x320 avec contrôleur AXS15231B.

Architecture\+:
\begin{DoxyItemize}
\item Pattern State Machine pour la navigation entre pages
\item Page\+Manager gère automatiquement l\textquotesingle{}affichage et les interactions
\item Chaque page hérite de la classe abstraite Page
\item Interface tactile I2C pour la navigation
\end{DoxyItemize}

Pages disponibles\+:
\begin{DoxyItemize}
\item Home\+Page\+: Page d\textquotesingle{}accueil avec informations système
\item Menu\+Page\+: Menu de navigation principal
\item Settings\+Page\+: Configuration de l\textquotesingle{}application
\item Status\+Page\+: Statut système en temps réel
\end{DoxyItemize}

Matériel\+:
\begin{DoxyItemize}
\item MCU\+: ESP32-\/\+S3 (Dual-\/core Xtensa LX7, 240MHz)
\item Écran\+: C3248\+W535\+C\+\_\+\+I\+\_\+Y (480x320, QSPI, RGB565)
\item Touch\+: Contrôleur I2C @ 0x3B
\item RAM\+: 8MB PSRAM intégrée
\end{DoxyItemize}

Séquence d\textquotesingle{}initialisation\+:
\begin{DoxyEnumerate}

\item Initialisation de l\textquotesingle{}écran QSPI et configuration en mode paysage
